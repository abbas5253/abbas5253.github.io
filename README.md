## Viewing

Visit the live site at: `https://[your-username].github.io/[repo-name]/`

## Printing to PDF

1. Open the resume in your browser
2. Press Cmd+P (Mac) or Ctrl+P (Windows)
3. Select "Save as PDF"
4. Uncheck "Headers and footers"
5. Save your PDF
